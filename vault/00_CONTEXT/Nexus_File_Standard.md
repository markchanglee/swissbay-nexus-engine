# Nexus File Standard

## Required sections

Every important Nexus file should include:

- Purpose
- Business Value
- Owner
- Inputs
- Outputs
- Workflow
- AI Support
- Related Documents
- Version History

## Quality test

A file is only useful if it:
- is specific to Swissbay
- creates business value
- can be used by a new employee
- links to related Nexus files
- supports current operations and future scale
