# Nexus Constitution

## Principles

1. Knowledge belongs to Swissbay.
2. AI prepares, humans approve.
3. Every process needs an owner.
4. If it happens twice, document it.
5. If it is not in CRM, it does not exist.
6. Build for today, architect for tomorrow.
7. Every file must create business value.

## Approval model

Every file ends with:

- Approve
- Revise
- Reject
