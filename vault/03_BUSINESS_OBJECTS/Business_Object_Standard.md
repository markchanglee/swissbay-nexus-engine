# Business Object Standard

## Purpose

Business objects define the shared language of Swissbay Nexus.

## Standard object structure

A business object should define:

- Purpose
- Business Value
- Owner
- Core Fields
- Statuses
- Relationships
- Workflows Using This Object
- AI Support
- Related Documents
- Version History
