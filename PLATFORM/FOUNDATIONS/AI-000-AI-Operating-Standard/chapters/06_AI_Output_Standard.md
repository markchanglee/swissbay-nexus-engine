# 06 — AI Output Standard

## Purpose

AI output should be clear, reviewable and traceable.

## Output Requirements

AI output should include:

- purpose
- assumptions
- source context used
- recommendation or draft
- risks
- next actions
- review requirement

## Draft Labelling

AI-generated content should be marked as draft until reviewed.

Recommended label:

```text
Status: AI Draft
```

## Traceability

Where possible, AI output should mention which Nexus files or sources it used.
