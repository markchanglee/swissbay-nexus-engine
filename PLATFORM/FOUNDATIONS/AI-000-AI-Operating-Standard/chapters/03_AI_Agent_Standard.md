# 03 — AI Agent Standard

## Purpose

Every AI agent must have a clear role, owner, boundaries and escalation path.

## Required Agent Sections

Every AI Agent Specification must include:

- Agent ID
- Agent Name
- Purpose
- Business Owner
- Allowed Inputs
- Allowed Outputs
- Context Sources
- Tools Allowed
- Prohibited Actions
- Escalation Rules
- Review Rules
- Version History

## Example Agent IDs

```text
AI-001 CEO Advisor Agent
AI-002 Sales Director Agent
AI-003 Procurement Manager Agent
AI-004 Finance Analyst Agent
AI-005 Marketing Agent
```

## Rule

An AI agent without an owner is not allowed to operate in production.
