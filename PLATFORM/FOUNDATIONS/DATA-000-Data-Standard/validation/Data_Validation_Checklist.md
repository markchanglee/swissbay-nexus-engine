# Data Validation Checklist

Use this checklist when reviewing a Business Specification.

- [ ] All important entities have IDs.
- [ ] Field types are defined.
- [ ] Required fields are marked.
- [ ] Status values are controlled.
- [ ] Relationships are explicit.
- [ ] Currency fields include currency.
- [ ] Dates use ISO format.
- [ ] Data owners are defined.
- [ ] Validation rules are documented.
- [ ] AI usage does not depend on undefined fields.
