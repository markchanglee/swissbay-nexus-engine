# 13 — Dashboards

---

# Overview

The Project Dashboard Portfolio provides operational, tactical and executive visibility into projects across the enterprise.

Dashboards transform governed Project information into actionable intelligence that supports project delivery, portfolio management, executive oversight and Artificial Intelligence.

Dashboards consume Project information.

They never own or modify Project data.

---

# Purpose

Project dashboards exist to:

- monitor delivery
- improve visibility
- strengthen governance
- support decision-making
- identify risks
- improve forecasting
- support portfolio management

Dashboards answer business questions rather than simply displaying data.

# DASH-PROJ-001 — Project Overview Dashboard

## Purpose

Provides operational visibility into an individual Project.

### Business Questions

- Is the Project on schedule?
- Is the budget under control?
- Which milestones are due?
- What are the active risks?
- Which deliverables remain outstanding?

### KPIs

- Project Health
- Schedule Performance
- Budget Performance
- Milestone Completion
- Risk Count

---

# DASH-PROJ-002 — Portfolio Dashboard

# AI Integration

Artificial Intelligence enhances dashboards by:

- forecasting outcomes
- identifying risks
- highlighting anomalies
- recommending actions
- generating executive summaries

AI explanations should accompany recommendations.

---

# Dashboard Governance

Every dashboard should:

- consume governed data
- display governed KPIs
- respect role-based security
- support auditability
- maintain consistent definitions

Dashboards never redefine business calculations.

# Dashboard Summary

The Project Dashboard Portfolio transforms governed project information into enterprise intelligence.

By combining operational visibility, executive reporting, financial oversight, resource planning, risk monitoring and AI-powered insights, Swissbay Nexus enables organisations to deliver projects with greater confidence, transparency and governance while maintaining complete traceability to the Project Business Object.
