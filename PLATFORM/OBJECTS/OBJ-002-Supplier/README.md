# OBJ-002 — Supplier

## Overview

The Supplier Business Object defines how Swissbay identifies, evaluates, approves, manages and monitors the organisations that provide products, services or capabilities to the business.

Suppliers are a core part of Swissbay’s operating model because customer value depends on reliable supply, competitive pricing, quality products, accurate fulfilment and strong supplier relationships.

## Purpose

The purpose of the Supplier object is to create one governed definition of a Supplier across Swissbay Nexus.

This object ensures that Procurement, Sales, Finance, Operations, Management and AI Agents all work from the same trusted supplier information.
